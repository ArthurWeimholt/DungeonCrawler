# DungeonCrawler
Proof of Concept Dungeon Crawler Game using ncurses library
---

## Dependencies
On Mac:
```
brew install sdl2 sdl2_mixer ncurses
```
On Debian/Ubuntu:
```
sudo apt-get update
sudo apt-get install build-essential libsdl2-dev libsdl2-mixer-dev libncurses5-dev
```

## Build

Run `make clean build`
